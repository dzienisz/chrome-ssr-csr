# Chrome Web Store Listing - v3.3.0

## Short Description (132 characters max)
Detect if websites use Server-Side Rendering (SSR) or Client-Side Rendering (CSR) with performance insights and analytics.

## Detailed Description

**Instantly analyze any website's rendering strategy!**

CSR vs SSR Detector helps developers, SEO specialists, and web enthusiasts understand how websites are built and perform.

### üéØ Key Features

**Rendering Detection:**
- Accurately detects Server-Side Rendering (SSR)
- Identifies Client-Side Rendering (CSR)
- Recognizes Hybrid/Islands Architecture
- Confidence scoring for each detection

**Performance Insights:**
- Core Web Vitals (LCP, CLS, FID)
- Time to First Byte (TTFB)
- Time to Interactive (TTI)
- Total Blocking Time (TBT)
- Resource metrics and cache analysis

**Framework Detection:**
- React, Next.js, Gatsby, Remix
- Vue, Nuxt
- Angular
- Svelte, SvelteKit
- Astro, Qwik, SolidJS
- WordPress, Shopify, Webflow
- And 15+ more frameworks!

**Page Analysis:**
- Automatic page type detection (e-commerce, blog, docs, app)
- Analytics tool identification
- PWA support detection
- Device and connection insights
